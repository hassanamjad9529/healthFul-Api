Welcome to the Blood Report Parser API!

Usage

Use the /upload endpoint to upload PDF or image files containing blood test reports.

Method

POST

Endpoint

/upload

Parameters

  • file: Required. One or more files to be uploaded. Accepted formats: PDF, PNG, JPG, JPEG.

Example Request

You can use the following curl command to test the upload:

curl -X POST -F 'file=@path/to/your/file.pdf' http://:/upload
